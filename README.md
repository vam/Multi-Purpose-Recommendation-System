# Multi-Purpose Recommendation System

## Overview

The Multi-Purpose Recommendation System is a Streamlit-based web app designed to provide top 10 personalized recommendations based on user preferences. Leveraging TF-IDF and count vectorization techniques, this app delivers tailored suggestions across various product categories, including movies, series, and books. The system is built to be adaptable and can be extended to accommodate additional product types.

## Features

- **Personalized Recommendations:** Offers top 10 recommendations tailored to user preferences.
- **Versatile Categories:** Supports multiple product categories such as movies, series, and books.
- **Extensible:** Easily extendable to include other product types.
- **Streamlit Interface:** User-friendly interface for seamless interaction.

## Technologies Used

- **Streamlit:** For building the web app interface.
- **TF-IDF:** To analyze and vectorize text data for recommendations.
- **Count Vectorization:** To convert text into a format suitable for model input.
